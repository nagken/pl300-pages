# Microsoft Learn summaries (PL-300)

One-line distillations of each Learn module.

- **Get data**: Sources, storage modes, gateway, semantic model vs local model.
- **Clean data**: Profile columns, fix nulls/errors, type conversions.
- **Transform/load**: Power Query - replace, profile, keys, types, append/merge, naming, folding.
- **Model design**: Relationships, RLS/OLS, role-playing dims, Date table.
- **DAX**: Measures, calc cols, time intel, iterators, VAR, what-if.
- **Optimize**: Drop unused, types, agg tables, Performance Analyzer.
- **Reports**: Choose visuals, format, theme, conditional fmt, slicers/filters, drillthrough, tooltips, bookmarks, paginated, accessibility, mobile.
- **Storytelling**: Bookmarks, custom tooltips, interactions, nav, sort, sync slicers, selection pane, drillthrough, export, mobile layout.
- **Patterns/trends**: Key Influencers, Decomp Tree, Q&A, grouping/binning/clustering, anomaly, time series, Analyze feature.
- **Workspaces/assets**: Roles, apps, dashboards, subscriptions, alerts, promote/certify.
- **Semantic models**: Gateway, refresh, RLS group membership, access.
