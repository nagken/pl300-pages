# PL-300 Glossary

| Term | Definition |
| --- | --- |
| **Semantic model** | Data model in Power BI (was 'dataset'). |
| **Power Query** | ETL editor using M language. |
| **M language** | Functional language for data shaping in Power Query. |
| **DAX** | Data Analysis Expressions; modeling and calc language. |
| **Import mode** | Data loaded in-memory into VertiPaq. |
| **DirectQuery** | Queries source live at interaction time. |
| **Dual** | Mixed mode in composite models. |
| **Composite model** | Mix of Import + DirectQuery tables. |
| **Star schema** | Fact + dimensions data modeling pattern. |
| **Measure** | Dynamic DAX aggregation. |
| **Calculated column** | DAX expression evaluated per row, stored in model. |
| **VAR** | Stored intermediate result in DAX. |
| **Time intelligence** | DAX functions over a Date table (TOTALYTD, DATEADD). |
| **RLS** | Row-Level Security - filter rows by role. |
| **OLS** | Object-Level Security - hide columns/tables by role. |
| **Workspace** | Collab container in Power BI service. |
| **App** | Polished published workspace content. |
| **Gateway** | Cloud-to-on-prem connector. |
| **Query folding** | Push transforms to source as SQL. |
| **Paginated report** | RDL-based, multi-page, print-ready. |
| **Promoted** | Owner-level endorsement of dataset/dataflow. |
| **Certified** | Admin-level formal endorsement. |
| **Deployment pipeline** | Dev/Test/Prod release flow (Premium/PPU). |
