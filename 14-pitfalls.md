# PL-300 Pitfalls

Mistakes that lose easy points on the exam.

- Breaking query folding via non-foldable Power Query steps.
- Choosing DirectQuery without understanding DAX/perf limits.
- Bidirectional cross-filter everywhere - ambiguity + perf.
- Calc columns where measures would suffice.
- Forgetting a marked Date table - time intelligence fails.
- Using personal gateway in production.
- Forgetting to add RLS users to role groups in the service.
- Sharing via workspace membership instead of App.
- Pie chart with >5 slices.
- Inaccessible color choices + no alt text.
